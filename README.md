# Simple Login Page

A lightweight static login page built with semantic HTML, responsive CSS, and a tiny JavaScript helper that simulates form handling.

## Features

- Centered card layout with a gradient-backed background
- Accessible form labels, status messaging, and keyboard-friendly controls
- Responsive spacing for small screens
- Non-functional form that simply demonstrates structure; replace the script with a real call when needed

## Usage

1. Clone the repository.
2. Open `index.html` in your browser.
3. The form only simulates a submission—open your browser console to see the logged input.
